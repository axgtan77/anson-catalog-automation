- **Positioned in top-right corner** (`top: 10px; right: 10px`)

- **Hover effect** - scales up slightly

### Popup:

- Semi-transparent overlay background

### Images:

### Product Info:
- Product name styling (blue, bold, uppercase)
- Price blocks with main/pack/case pricing
- Details text styling
- Description (shows only in popup)
